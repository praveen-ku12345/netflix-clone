# Netflix Clone

A fully functional Netflix clone built using **HTML**, **CSS**, **JavaScript**, **Node.js**, **React.js**, and **SQL**. This project replicates the basic features of Netflix, allowing users to browse, search, and manage movies and TV shows. It aims to provide a similar user experience to the original platform.

## Features

- **User Authentication**: Sign up, login, and manage user profiles.
- **Movie/TV Show Browser**: Browse through a list of movies and TV shows categorized by genre.
- **Search Functionality**: Search for movies or TV shows using a search bar.
- **Movie Details Page**: View detailed information about a movie/TV show, including description, release year, and cast.
- **Watch Trailer**: Play the trailer of the selected movie or TV show.
- **User Profile**: Users can manage their personal profiles and preferences.

## Tech Stack

- **Frontend**:
  - HTML5, CSS3, JavaScript
  - React.js for building interactive UI components
  - React Router for navigation
  - Axios for API calls

- **Backend**:
  - TMDB
  - SQL database (e.g., MySQL) for storing user and movie data

- **Authentication**:
  - Firebase for secure user authentication

- **Media**:
  - Embedded video player for watching trailers
