# Netflix Clone

A responsive movie streaming platform that dynamically fetches and displays movies and trailers using the TMDB API. This project was developed using React.js for the front-end and includes dynamic routing for seamless navigation between pages.

## Features

- **Dynamic Content**: Fetch and display movies in categories like Popular Movies, Top Rated, and Upcoming Movies.
- **Trailer Playback**: View movie trailers hosted on YouTube by clicking on movie posters.
- **Responsive Design**: Optimized for all screen sizes to ensure a great user experience on both desktop and mobile devices.
- **Dynamic Routing**: Navigate between pages smoothly without reloading.

## Tech Stack

- **Front-end**: HTML, CSS, JavaScript, React.js
- **Firebase**: for UserAuthentication
- **Database**: SQL 
- **API Integration**: TMDB API for movie data and trailers
